# ROHITH SILAGANI | Portfolio v1.0

A high-performance, production-grade portfolio designed for **ROHITH SILAGANI**, Data Analyst.

## Tech Stack
- **Framework**: React + Vite
- **Styling**: TailwindCSS (Swiss Modernist Archetype)
- **Animation**: GSAP, Framer Motion
- **Scrolling**: Lenis (Smooth Scroll)
- **Icons**: Lucide React

## Key Features
- **The Analyst Persona**: Every element reflects precision, logic, and insight.
- **Custom SVG Engine**: All illustrations are generated via inline SVG (no stock images).
- **Interactive Data Chart**: Radial visualization of skills with hover-state diagnostics.
- **Mechanical Motion**: Robotic snaps and precise transitions (no bounce easing).
- **Easter Egg**: Hidden currency forecasting secret in the bottom-left corner.

## Project Structure
- `/src/sections`: Main page sections (Hero, About, etc.)
- `/src/components`: Reusable components (CustomCursor, etc.)
- `/src/data`: Centralized content store (`content.js`)
- `/src/assets`: Static assets

## How to Update Content
To change any text, project, or skill, simply modify the `src/data/content.js` file. The UI will automatically reflect the changes.

## Deployment
This project is ready for deployment on Vercel. 
1. Push to GitHub.
2. Connect to Vercel.
3. Use `npm run build` as the build command.
4. Output directory: `dist`.
